# Description.

This project create a palette with 32768 unique colors. Each color contains three base color components: red, green, and blue. 
In each cell, there is a number shows how many unique colors have been shown on the palette.

## Set up the dependencies

### `npm i`

## Run the project

### `npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
